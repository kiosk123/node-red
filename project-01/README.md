# 설치 및 실행

### 1. nodejs를 설치한다.
### 2. node-red를 설치한다.
```bash
# 글로벌 설치
$ npm install -g -unsafe-perm node-red

# 프로젝트 폴더로 이동 후 명령어 실행 - 선택사항 : node-red 실행 후 대시보드에서 설치해도됨
$ cd project-01
$ npm install node-red-dashboard
```

### 3. node-red 실행
```
$ node-red
```

### 4. node-red 접속
`http://localhost:1880`으로 접속
